# RailsMarkdown
Short description and motivation.

## 依赖
* libgit2
  * MacOS: `brew install libgit2`
* cmake

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'rails_markdown'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install rails_markdown
```
