# RailsMarkdown

用 Markdown 写作，同步到应用，微信公众号，知乎等地方。


## 依赖
* libgit2
  * MacOS: `brew install libgit2`
* cmake

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'rails_markdown'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install rails_markdown
```
